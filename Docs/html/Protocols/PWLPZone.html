PWLPZone Protocol Reference

Conforms to NSObject
Declared in PWLPZoneProtocol.h

Overview

This protocol represents the zone data generated by an object which implements the PWLPZone protocol. This object incorporates a name, identifier, code, tags and several other properties that can be useful to query. Typical implementations will append additional data to zone object.
