fetchLocationsWithCompletionHandler:

@discussion Fetches the complete set of locations from the Location Marketing server and calls the given completion handler.

- (void)fetchLocationsWithCompletionHandler:(void ( ^ ) ( UIBackgroundFetchResult ))completionHandler

Parameters

completionHandler

Block that will be executed when the fectch is complete. The fetch result paramter can have the following values:

UIBackgroundFetchResultNewData if a successful call to Location Marketing was made. UIBackgroundFetchResultNoData if locations are not out of sync. UIBackgroundFetchResultFailed if there is a problem.
