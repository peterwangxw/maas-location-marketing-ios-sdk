identifier

The identifier the SDK uses.

@property (nonatomic, readonly) NSString *identifier

Discussion

The type of identifier can be IDFA or IDFV and the format is similar to ‘00000000-0000-0000-0000-000000000000’.

IDFA is applied when AdSupport.framework is included in the project and IDFV is applied when it is not.

identifierType

The type of identifier.

@property (nonatomic, readonly) NSString *identifierType

Discussion

The value can be IDFA or IDFV.

IDFA is the identifier for advertisers applied when AdSupport.framework is included in the project. IDFV is the identifier for vendors applied when AdSupport.framework is NOT included in the project.
