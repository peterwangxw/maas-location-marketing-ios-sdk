PWLPLocationUpdatesManagerDelegate Protocol Reference

Conforms to CLLocationManagerDelegate
Declared in PWLPLocationUpdatesManagerDelegateProtocol.h

Overview

PWLocationUpdatesManagerDelegate objects will receive the same callbacks as CLLocationManagerDelegate objects and also have the option to specify some desired values for the PWLocationUpdatesManager configuration.
